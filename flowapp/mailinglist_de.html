FASD Survey: Mailingliste

 

Um sich in die Mailingliste 'fasd@webboard.ethz.ch' einzutragen, müssen Sie wie folgt vorgehen:

  1. Schreiben Sie sich im Webboard Formular ein. Zum Einschreiben müssen Sie einen Login-Namen, Ihre Namen sowie Ihre e-Mail Adresse angeben.
  2. An die angegebene Adresse wird Ihnen daraufhin Ihr Passwort zugeschickt.
  3. Mit diesem Passwort und Ihrem Login-Namen können Sie nun in das Webboard einloggen.
  4. Nachdem Sie sich im Webboard eingeloggt haben, müssen Sie sich in der FASD-Mailingliste anmelden. Mit einem Klick auf "More..." können Sie die notwendigen Optionen für diesen Zweck einblenden. Klicken Sie daraufhin auf "Mailing Lists", um die Übersicht der Mailinglisten anzuzeigen.
  5. Wählen Sie nun aus den Mailinglisten "Conference FASD".
  6. Gehen Sie danach nochmals auf die "More Options"-Seite, klicken Sie auf "Email Notification" und selektieren Sie "Conference FASD". Damit werden Mails an die FASD-Mailingliste automatisch an Sie weitergeleitet.
